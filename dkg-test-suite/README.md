## Prepare tests:

1. Run `cd dkg-test-suite`
2. Run `yarn install`
3. Install [ts-node](https://typestrong.org/ts-node/)

# Anchor Update tests:

From terminal 1:

1. Run `./scripts/run-standalone.sh`
2. Wait until Keygen completes

From terminal 2:

1. Run `ts-node https://typestrong.org/ts-node/` for anchor proposal tests
